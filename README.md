# Tata Steel Onboarding Process Page

A simple and responsive onboarding webpage created for the Tata Steel SNTI internship onboarding process.

This project is a single-page HTML website designed to display onboarding-related information in a clean and user-friendly format.

The webpage includes:

- Structured onboarding information
- Simple and responsive design
- Easy navigation and readability

## ⭐ GitHub Pages Hosting

This project is hosted using GitHub Pages.

To deploy:

1. Upload project files to GitHub repository
2. Go to Settings → Pages
3. Select:
   - Source → Deploy from branch
   - Branch → main
   - Folder → /root
4. Save changes
5. Website becomes live automatically
